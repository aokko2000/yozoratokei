# 夜空時計 (yozoratokei)

「空にかざすと、その方向の星座・月が幻想的に浮かぶ」M5Stack CoreS3 デバイス。
色時計 [iroirotokei](https://github.com/aokko2000/iroirotokei) の姉妹機。設計メモは [PLAN.md](PLAN.md)。

## 機能

### 小さなプラネタリウム (星空モード)
- 空にかざすと、その方向の**恒星113個・主要18星座**が画面に浮かぶ
- 星座線 + 星座名 (画面中央にとらえた星座は図鑑ふうに大きく表示)
- 有名な星 (ベガ・シリウス・北極星・すばる など19個) に名前ラベル
- 星は等級に応じたグローとまたたき、地平線には東西南北マーカー
- 空にも**月が実位置・実位相**で浮かぶ
- 傾き補償コンパス (BMM150磁気 + BMI270加速度)。上部に方位・仰角を常時表示
- **8の字キャリブレーション**: 初回起動時に自動開始。オフセットはNVSに保存され次回から不要

### 月時計モード (DinBase据え置き用)
- 大きな月 (暈つき・欠け際まで正確) + **月齢・月名・輝面比・満月/新月まで**
- **月食**カウントダウン (2026-03-03 皆既月食など)、月食中は月が赤銅色に
- **流星群**情報 (活動中/次の極大)
- 大きな現在時計 + 日付・曜日

### 月齢の精度
- 月齢は **Meeusの新月公式**で「直前の新月からの経過日数」として算出 (誤差 数分)。カレンダー・月齢アプリと一致。

### 表示プリセット (外でも見やすく)
- **標準** / **屋外** (高コントラスト・最大輝度) / **夜間** (赤・暗順応を保つ)

### 時刻
- RTC + **手動時刻合わせ** (月時計モードで Scroll ボタン長押し → 時刻合わせ画面)。RTCに保存され次回も保持。
- シリアル(115200)から `D 2026-07-20 21:30:00` / `T 21:30:00` でも設定可

## 操作

| 操作 | 動作 |
|---|---|
| 画面タップ | プラネタリウム ⇄ 月時計 |
| 星空で画面長押し(1秒) | コンパス再キャリブレーション |
| 月時計で画面長押し | 音 ON/OFF |

### Scroll Unit (Port A・任意)
挿すと使える回転ノブ。**クリックで対象を切替** (時間→星座→画面→音量)、**回して操作**、**長押しでリセット** (月時計では長押しで時刻合わせ画面)。
- **時間**: 星空を早送り/巻き戻し (タイムトラベル。星と月が動く)
- **星座**: 星座を順に選択 → 中央から「その星座はこっち」の矢印で誘導
- **画面**: 表示プリセット切替 / **音量**: 調節

### サウンド
iroirotokei譲りのペンタトニック星空音。星座を中央にとらえると星座ごとのモチーフ、明るい星できらめき音、キャリブレーション完了/毎正時に鐘のチャイム。

## ハードウェア

- **M5Stack CoreS3** (ESP32-S3, 2.0" IPS 320×240, BMI270+BMM150, RTC BM8563, スピーカー)
- 任意: **Scroll Unit** (Port A)、DinBase (据え置き)

## ビルド・書き込み (VSCode + PlatformIO)

1. VSCode でこのフォルダ (`C:\yozoratokei`) を開く
2. 下部ステータスバーの **✓ (Build)** → コンパイル
3. CoreS3 をUSB-Cで接続し、**→ (Upload)** → 書き込み
4. シリアルモニタは **🔌 (Serial Monitor)**、115200bps

## 設定 (src/main.cpp 冒頭)

- `MAG_DECLINATION_DEG`: 磁気偏角。東京近辺は約 -8°(西偏)。地域に合わせて調整
- `OBS_LAT_DEG` / `OBS_LON_DEG`: 観測地 (既定: 東京)
- `SIGN_AX..SIGN_MZ` / `SCREEN_SX/SY`: 実機で方位・仰角・画面の向きが合わない場合の軸符号調整用
