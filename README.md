# 夜空時計 (yozoratokei)

「空にかざすと、その方向の星座・月が幻想的に浮かぶ」M5Stack CoreS3 デバイス。
色時計 [iroirotokei](https://github.com/aokko2000/iroirotokei) の姉妹機。設計メモは [PLAN.md](PLAN.md)。

## 現在の状態 (ステップ4まで)

- **小さなプラネタリウム**: 空にかざすと、その方向の恒星113個・主要18星座が画面に浮かぶ
  - 星座線 + 星座名 (画面中央にとらえた星座は図鑑ふうに大きく表示)
  - 有名な星 (ベガ・シリウス・北極星・すばる など19個) に名前ラベル
  - 星は等級に応じたグローとまたたき、地平線には東西南北マーカー
- 傾き補償コンパス: BMM150(磁気) + BMI270(加速度)。上部に方位・仰角を常時表示
- 8の字キャリブレーション: 初回起動時に自動開始。オフセットはNVSに保存され次回から不要
- 時刻: RTC + 起動時に保存済みWi-FiでNTP自動同期 (iroirotokei移植)。未設定時はビルド時刻で概算
  - シリアル(115200)から `N`=NTP同期 / `D 2026-07-20 21:30:00` / `T 21:30:00` で設定可
- **月時計モード** (DinBase据え置き用): 大きな月(暈つき・欠け際まで正確)+月齢・月名・輝面比・満月/新月まで、月食カウントダウン、流星群情報、時計表示 (iroirotokei移植)
- プラネタリウム側の空にも月が実位置・実位相で浮かぶ
- 操作: **タップ** でプラネタリウム ⇄ 月時計 / 画面**長押し(1秒)** で再キャリブレーション / シリアルで `V` =デバッグ表示

## ビルド・書き込み (VSCode + PlatformIO)

1. VSCode でこのフォルダ (`C:\yozoratokei`) を開く
2. 下部ステータスバーの **✓ (Build)** をクリック → コンパイル
3. CoreS3 をUSB-Cで接続し、**→ (Upload)** をクリック → 書き込み
4. シリアルモニタは **🔌 (Serial Monitor)**、115200bps

## 設定 (src/main.cpp 冒頭)

- `MAG_DECLINATION_DEG`: 磁気偏角。東京近辺は約 -8°(西偏)。地域に合わせて調整
- `SIGN_AX..SIGN_MZ`: 実機で方位・仰角が合わない場合の軸符号調整用
